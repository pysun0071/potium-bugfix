#01 점프 후 발판 설치 시 더블 점프 발생
PHYSICS ▼
현상
점프 중 Z키로 발판을 설치하면 공중에서 추가 점프가 가능해지는 버그
원인
발판 타일 생성 시 Collider2D가 즉시 생성되면서 OnCollisionEnter2D가 발동, isJumping = false로 리셋되어 공중에서 재점프 가능해짐
해결
공중에서 발판 설치 시(isJumping = true)에만 isJustPlaced = true 플래그를 켜서 OnCollisionEnter2D 착지 판정을 0.3초간 차단
#02 착지 시 애니메이션 간헐적 멈춤
PHYSICS ▼
현상
발판 위를 걷다가 타일 이음새에서 점프 애니메이션이 멈추는 현상
원인
OnCollisionEnter2D만 사용하여 착지 감지 시, 타일 경계에서 충돌이 순간 끊겼다 재연결될 때 isJumping = true가 유지됨
해결
OnCollisionStay2D 추가로 발판 위에 있는 동안 매 프레임 착지 상태 유지. 단, velocity.y <= 0.1f일 때만 착지로 인정하여 점프 직후 오판 방지
#03 점프가 전혀 안 되는 현상
PHYSICS ▼
현상
스페이스바를 눌러도 플레이어가 점프하지 않음. 콘솔에 [Jump] Executed는 찍힘
원인
Inspector에서 Jump Power 값이 0.5로 설정되어 있어 힘이 너무 약해 눈에 보이지 않았음
해결
Inspector → PlayerStageMove → Jump Power 값을 0.5에서 12로 수정
#04 PlayerStageMove 상속 구조에서 점프 불가
PHYSICS SCRIPT ▼
현상
PlayerMove를 상속한 PlayerStageMove 구조에서 점프가 동작하지 않음
원인
Unity에서 OnCollisionEnter2D 등 이벤트 함수는 상속 시 자식 클래스가 있으면 부모 것이 자동 호출되지 않아 착지 감지가 전혀 안 됨
해결
상속 구조를 폐기하고 PlayerMove 코드를 PlayerStageMove에 직접 복붙 후 발판 설치 코드만 추가하는 방식으로 변경
#05 발판 위에서 점프 시 너무 높이 뜀
PHYSICS ▼
현상
점프 후 공중에서 OnCollisionStay2D가 즉시 isJumping = false로 리셋, 연속 점프가 되어 매우 높이 뜸
원인
점프 직후 플레이어가 아직 발판 위에 있는 상태에서 OnCollisionStay2D가 즉각 착지 판정을 내려버림
해결
점프 입력 시 isJustJumped = true 플래그를 0.2초간 켜서 해당 시간 동안 착지 판정 차단
#06 좌우 이동이 안 되는 현상
PHYSICS ▼

PlayerStageMove 교체 후 좌우 이동이 전혀 동작하지 않음
원인

#07 발판 설치 시 애니메이션 멈춤
PHYSICS ▼
현상
Z키로 발판 설치 후 캐릭터 애니메이션이 점프 포즈에서 멈춤
원인
isJustPlaced 타이머가 바닥에서 설치 시에도 착지 판정을 차단하여 isJumping이 해제되지 않음

#08 고스트 스프라이트 위치와 실제 설치 위치 불일치
UI ▼
현상
미리보기 고스트가 보이는 위치와 실제 Z키 설치 위치가 다름
원인
고스트는 X좌표를 플레이어 실제 위치로, 설치는 GetCellCenterWorld()의 셀 중앙 X로 계산하여 좌표 불일치 발생
해결
고스트 위치를 설치 위치와 동일하게 GetCellCenterWorld()로 통일 + Vector3.Lerp로 부드럽게 이동. 처음 나타날 때는 즉시 위치로 점프
#09 고스트 스프라이트 뚝뚝 끊김
UI ▼
현상
플레이어 이동 시 고스트가 셀 단위로 뚝뚝 끊겨서 이동함
원인
고스트 위치를 GetCellCenterWorld()로 매 프레임 직접 지정하여 타일 그리드 단위로 점프하듯 이동
해결
Vector3.Lerp(현재위치, 목표위치, Time.deltaTime * 25f)로 보간하여 부드럽게 이동

#13 발판이 배경 뒤로 숨는 현상

#15 D3D11 GPU Timeout으로 Unity 에디터 강제 종료
UNITY ▼
현상
"Failed to present D3D11 swapchain due to device reset/removed" 에러로 Unity 에디터 강제 종료
원인
GPU 과부하로 Windows TDR(Timeout Detection and Recovery)이 발동하여 드라이버 강제 리셋
해결
Edit → Project Settings → Player → Other Settings → Auto Graphics API for Windows 체크 해제 → Direct3D11 제거 → OpenGLCore 추가 후 Unity 재시작
